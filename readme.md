 Built an AI-Based Drone Delivery Optimization System

I recently developed a system that goes beyond simple prediction and actually makes real-world delivery decisions.

🔹 What it does:

* Assigns deliveries to drones intelligently
* Optimizes delivery time and cost
* Considers battery constraints
* Suggests charging stops during delivery
* Provides a route timeline like real delivery platforms
⚠️ Random Forest model file is not included due to size constraints.
Run `python src/model_training.py` to generate it.
🔹 Tech used:
Python, Streamlit, Machine Learning (KMeans & Random Forest)

This project helped me understand how AI can be applied to real logistics problems rather than just building models.
